# getGrade

Write a function that takes a grade percentage as input and displays the corresponding letter grade.

| Percentage | Grade |
|------------|-------|
| >= 80      | A     |
| 70 - 79     | B     |
| 60 - 69     | C     |
| 50 - 59     | D     |
| < 50        | F     |

```js
example 1
input: 75
output: 'B'

example 2
input: 80
output: 'A'

example 3
input: 45
output: 'F'
```
