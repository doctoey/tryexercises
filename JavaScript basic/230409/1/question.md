**ageClassification**

Write a function that takes the user's age as input and return a message indicating whether they are a minor, an adult, or a senior citizen.

| Age Range      | Category |
|----------------|----------|
| age <= 18      | Minor    |
| 18 < age < 65  | Adult    |
| age >= 65      | Senior   |

  ```js
  example 1
  input: 25
  output: 'Adult'

  example 2
  input: 70
  output: 'Senior'
  ```
